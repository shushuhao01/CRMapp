# TabBar 图标说明

需要准备以下图标文件（建议尺寸 81x81 像素）：

## 未选中状态（灰色）
- home.png - 首页图标
- calls.png - 通话图标
- stats.png - 统计图标
- settings.png - 设置图标

## 选中状态（绿色 #34D399）
- home-active.png - 首页图标（选中）
- calls-active.png - 通话图标（选中）
- stats-active.png - 统计图标（选中）
- settings-active.png - 设置图标（选中）

## 图标设计建议
- 使用简洁的线性图标
- 未选中状态颜色：#999999
- 选中状态颜色：#34D399
- 图标线条粗细：2-3px
- 图标内容区域：约 60x60 像素，居中

## 推荐图标来源
- https://www.iconfont.cn/
- https://iconpark.oceanengine.com/
